# アルゴリズム(仕組み)
## htmlでボタン入力で変数を代入する仕組み
[参照記事はこちら](https://itsakura.com/js-textbox)
```html
<input type="button" value="(ボタンのキャプション)" onclick="(実行するJavaScriptのコード)"/>
```
これでボタンクリックでJavaScriptの関数が実行出来る。
